# spider-zhihu-hot

爬取知乎热榜

## 使用指南

1. 登录知乎
2. 浏览器安装 Cookie Editor
3. 之后使用 Cookie Editor export `.zhihu.com` 下的 cookie 到 `./env/cookies.json`
4. pnpm start
